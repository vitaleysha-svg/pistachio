# Module 03 - Goals, Patterns, and the ONE THING

## Outcome
The member defines a clear north star and uses AI to protect focus against distraction.

## Duration
45-60 minutes.

## Lessons
1. `lesson-01-the-one-thing.md`
2. `goals-template.md`
3. `patterns-template.md`
4. `exercise-weekly-planning.md`
5. `module-checklist.md`

## Assignment
Define one weekly ONE THING and one weekly "not doing" list, then run a full week using both.

## Pass Criteria
- ONE THING is documented for the week.
- 3 personal patterns identified.
- Weekly plan created with explicit non-goals.
