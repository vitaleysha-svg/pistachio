# Module 04 - Never Make The Same Mistake Twice

## Outcome
The member installs a learned-mistakes loop so the AI improves every week instead of repeating the same failure.

## Duration
30-40 minutes.

## Lessons
1. `lesson-01-fix-the-instruction-not-yourself.md`
2. `learned-mistakes-template.md`
3. `exercise-first-five-mistakes.md`
4. `module-checklist.md`

## Assignment
Document your first 5 mistakes in the template and run one before/after test where the new rule changes output quality.

## Pass Criteria
- 5 mistakes logged with exact trigger + rule.
- At least 1 proven quality improvement from a rule update.
- Member can explain the difference between "feedback" and "system update."
