# Exercise - Build Tomorrow's Brief Tonight

## Objective
Reduce startup friction and win the first hour of the day.

## Steps
1. Write tomorrow's ONE THING tonight.
2. Block two deep work windows.
3. List 3 priorities in execution order.
4. Define one task you will NOT do.
5. Save the brief where you can open it in one click.

## Done When
- Tomorrow's brief is pre-written.
- Priority order is clear.
- You removed at least one distraction.
