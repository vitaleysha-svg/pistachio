# Module 05 - The Morning Brief That Runs Your Day

## Outcome
The member sets up a daily morning brief that aligns schedule, priorities, and ONE THING before the day starts.

## Duration
30-40 minutes.

## Lessons
1. `lesson-01-brief-before-busywork.md`
2. `morning-brief-template.md`
3. `exercise-build-tomorrows-brief.md`
4. `module-checklist.md`

## Assignment
Run the morning brief for 3 consecutive days and post one decision that changed because of it.

## Pass Criteria
- Morning brief generated before work starts.
- ONE THING is explicit each day.
- At least one low-value task was removed because of the brief.
