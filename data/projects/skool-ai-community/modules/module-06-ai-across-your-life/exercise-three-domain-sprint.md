# Exercise - Three-Domain Sprint

## Objective
Prove you can run the same CoS system across multiple parts of life.

## Steps
1. Pick 3 domains where you're leaking time.
2. Build one workflow using the playbook template for each domain.
3. Run each workflow once this week.
4. Track one measurable improvement per domain.

## Good Examples
- Work: weekly planning + priority sequencing.
- Communication: hard email drafting in your voice.
- Decisions: one high-stakes choice with pros/cons and next action.

## Done When
- All 3 domains have active workflows.
- You can show one concrete win per domain.
