# Exercise - Build Three Skills

## Objective
Install a real quality layer you can reuse daily.

## Skills to build
1. Voice/style skill
2. Decision-support skill
3. Execution-checklist skill

## Steps
1. Define each skill trigger.
2. Write 3-5 rules per skill.
3. Run one real task with each skill active.
4. Log one improvement and one fix per skill.

## Done When
- 3 skills exist.
- Skills are actually used, not just stored.
- One output quality jump is documented.
