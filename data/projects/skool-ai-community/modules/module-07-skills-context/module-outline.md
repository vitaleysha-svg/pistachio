# Module 07 - Skills and Context Architecture

## Outcome
The member builds reusable skills and a context map so AI stops forgetting critical details between tasks.

## Duration
45-60 minutes.

## Lessons
1. `lesson-01-skills-vs-context.md`
2. `skill-template.md`
3. `context-map-template.md`
4. `exercise-build-three-skills.md`
5. `module-checklist.md`

## Assignment
Create 3 skills and 1 context map tied to your weekly priorities.

## Pass Criteria
- 3 skills created with clear triggers.
- Context map points to exact files.
- AI uses skill rules correctly on one real task.
