# Exercise - First Two-Agent Run

## Objective
Ship one real output using builder + reviewer roles.

## Steps
1. Define goal and ship criteria.
2. Assign Agent A (build) and Agent B (review).
3. Run builder output.
4. Run reviewer checks.
5. Resolve must-fix issues.
6. Ship with handoff notes.

## Done When
- Output shipped.
- Reviewer found at least one real issue.
- You documented one process improvement.
