# Module 10 - Multi-Agent Power Moves

## Outcome
The member can delegate one project across multiple agents with clear roles, non-overlapping scope, and review gates.

## Duration
45-60 minutes.

## Lessons
1. `lesson-01-agent-teams-not-agent-chaos.md`
2. `delegation-template.md`
3. `review-protocol.md`
4. `exercise-first-two-agent-run.md`
5. `module-checklist.md`

## Assignment
Run one two-agent workflow with a reviewer loop and ship one complete artifact.

## Pass Criteria
- Roles are clearly split.
- Reviewer catches at least one issue before final output.
- Final artifact is shipped with handoff notes.
