# Multi-Agent Review Protocol

## Required Checks
1. Fact check: no fabricated claims.
2. Scope check: no unauthorized changes.
3. Voice check: direct, human, not AI-generic.
4. Verification check: evidence attached.

## Severity Levels
- P0: Wrong facts / dangerous changes / scope breach.
- P1: Missing required output.
- P2: Quality polish issue.

## Rule
No final output ships with unresolved P0/P1.
