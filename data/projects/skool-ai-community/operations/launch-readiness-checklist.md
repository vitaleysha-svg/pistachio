# Launch Readiness Checklist

## P0 - Must Be True Before Launch
- [ ] Module 1 can be completed by a non-technical member without live rescue.
- [ ] 5-minute win path tested end-to-end.
- [ ] Starter kit files are complete and downloadable.
- [ ] Onboarding email sequence is loaded.
- [ ] Canonical email files are loaded (not overview doc).
- [ ] Community guidelines are published.
- [ ] Week 1 content calendar is pre-scheduled.
- [ ] KPI dashboard/tracker is ready before first member joins.
- [ ] `day-0-launch-runbook.md` owner/time blocks are assigned.

## P1 - Should Be True
- [ ] Module 2 and 3 walkthrough videos recorded.
- [ ] First live call agenda prepared.
- [ ] FAQ published.
- [ ] Founding member feedback form ready.

## P2 - Nice To Have
- [ ] Installer backlog reviewed.
- [ ] Advanced modules pre-recorded.
- [ ] Referral loop set up.

## No-Launch Rules
Do not launch if any P0 item is unchecked.
