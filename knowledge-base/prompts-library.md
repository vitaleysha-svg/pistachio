## Tested Prompts (Nano Banana)

### Base Portrait Prompt
```
Egyptian/Brazilian mix woman, 20-21 years old,
tooth gap (diastema), frizzy unstyled hair,
visible skin pores, slight blemishes,
tired bored expression, asymmetrical features,
full-frame DSLR, 50mm Prime lens f/1.4,
sub-surface scattering, natural ISO grain,
low-key tungsten lamp lighting,
messy bedroom background,
oversized vintage band t-shirt,
authentic clutter (soda cans, tangled wires),
gaming introvert aesthetic,
candid off-center framing
```

### Negative Prompt (What to Avoid)
```
perfect skin, symmetrical, airbrushed, plastic,
studio lighting, white background, professional model,
perfect hair, makeup, posed, staged,
oversaturated, HDR, smooth skin
```
